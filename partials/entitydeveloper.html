Entity Developer – Step by Step Útmutató

1. lépés – Letöltés és telepítés

  • Nyisd meg a https://www.devart.com/entitydeveloper/ oldalt
  • Kattints a Download gombra
  • Futtasd a telepítőt, majd kövesd a telepítési lépéseket

2. lépés – Integrálás Visual Studio-val

  • Visual Studio újraindítása után az Entity Developer elérhető lesz a menüben vagy jobbklikkel a Solution Explorer-ben
  • Ha nem látod: Tools → Extensions and Updates → keresd: Entity Developer

3. lépés – Új modell létrehozása

  1. Kattints jobb gombbal a projektedre → Add → New Item
  2. Válaszd: Devart Entity Model (pl. Model.edml)
  3. A varázslóban válaszd ki az adatbázis típust (pl. SQL Server)
  4. Állítsd be az adatkapcsolatot
  5. Válaszd ki az importálni kívánt táblákat

4. lépés – Kódgenerálás beállítása

  • A modell mentése után automatikusan generálódik a DbContext, entitások, stb.
  • A Properties panelben választhatod az ORM típust: EF Core, EF 6, LinqConnect, stb.
  • Output fájlok: *.cs a modellekhez és context-hez

5. lépés – Használat a kódban


  // Példa EF Core DbContext használat:
  using (var db = new MyDbContext())
  {
      var lista = db.Felhasznalok.ToList();
  }

6. lépés – Modell frissítése

  • Ha az adatbázis megváltozik: nyisd meg az .edml fájlt
  • Kattints: Update From Database → válaszd ki az új vagy módosult elemeket
  • Mentés után a kód automatikusan frissül

ℹ️ További infó és dokumentáció: Entity Developer Docs

SQL Server Management Studio – Step by Step Útmutató

1. lépés – SSMS letöltése és telepítése

  • Nyisd meg: https://aka.ms/ssms
  • Kattints a Download gombra, majd futtasd a telepítőt
  • Telepítés után indítsd el a SQL Server Management Studio-t

2. lépés – Kapcsolódás szerverhez

  1. Indításkor megjelenik a Connect to Server ablak
  2. Server type: Database Engine
  3. Server name: localhost vagy a szerver neve
  4. Authentication: Windows Authentication vagy SQL Server Authentication

3. lépés – Új adatbázis létrehozása

  1. Object Explorer → jobb klikk a Databases mappán
  2. Válaszd: New Database
  3. Adj nevet, majd kattints OK

4. lépés – Táblák létrehozása

  • Az adatbázis alatt jobb klikk → New Table
  • Hozz létre mezőket (pl. Id INT PRIMARY KEY, Nev NVARCHAR(100))
  • Mentés: Ctrl + S, majd add meg a tábla nevét

5. lépés – SQL utasítások futtatása

  1. Object Explorer → jobb klikk az adatbázison → New Query
  2. Írj be például egy SELECT vagy INSERT utasítást

  -- Tábla lekérdezése
  SELECT * FROM Felhasznalok;
  
  -- Új sor beszúrása
  INSERT INTO Felhasznalok (Id, Nev) VALUES (1, 'Teszt Elek');

6. lépés – Adatok böngészése

  • Object Explorer → jobb klikk egy táblán → Select Top 1000 Rows
  • A lekérdezés automatikusan generálódik

7. lépés – Felhasználók és jogosultságok


  -- Új SQL felhasználó létrehozása
  CREATE LOGIN tesztuser WITH PASSWORD = 'ErősJelszó123';
  
  -- Felhasználó hozzárendelése adatbázishoz
  USE TesztAdatbazis;
  CREATE USER tesztuser FOR LOGIN tesztuser;
  EXEC sp_addrolemember 'db_datareader', 'tesztuser';
  EXEC sp_addrolemember 'db_datawriter', 'tesztuser';

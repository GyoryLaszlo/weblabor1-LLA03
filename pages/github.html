Githubbal kapcsolatos segédletek

Github használata

A GitHub egy nyílt forráskódú verziókezelő rendszer, amely lehetővé teszi a kódok tárolását és megosztását. A használatához szükséges lépések:

  1. Regisztrálj egy fiókot a github.com oldalon.
  2. Hozz létre egy új repository-t a kódjaid számára.
  3. Klónozd a repository-t: git clone
  4. Add hozzá a változásokat: git add
  5. Commit: git commit
  6. Push: git push
  7. Pull request: kérj visszajelzést másoktól.
  8. Hibák jelentése: issue használata.
  9. Branch-ekkel párhuzamos fejlesztés.
  10. Merge: ágazatok összeolvasztása.
  11. Rebase: commit történet újraírása.
  12. Stash: ideiglenes mentés git stash.
  13. Tag-ek: kiadások megjelölése.
  14. Fork: más repository-ból saját példány létrehozása.
  15. Pull: frissítések letöltése git pull.
  16. Fetch: frissítések letöltése merge nélkül git fetch.
  17. Cherry-pick: egy commit másolása másik branch-re.
  18. Bisect: hibás commit megtalálása.
  19. Blame: soronkénti szerző információk.
  20. Reflog: helyi commit történet nyomon követése.
  21. Config: git beállítások kezelése git config.
  22. Remote: távoli repository-k kezelése.
  23. Submodule: másik repo beágyazása.
  24. Hooks: automatizált műveletek (pre-commit, pre-push, stb.).
  25. .gitignore: figyelmen kívül hagyandó fájlok.
  26. Git LFS: nagy fájlok kezelése.
  27. Git Flow: szabályozott fejlesztési modell.
  28. Git SVN: Subversion integráció.
  29. Git Archive: zip csomag készítése a repo-ból.
